# Episode Selector
Selects a random episode from the given selection.

## Setup
You'll need to create a spotify app to use the Web API. Then get the *Client ID* and *Client Secret* for the app. Follow the link below for details.

https://developer.spotify.com/documentation/web-api/tutorials/getting-started

```sh
# Set env vars
cp .env.example .env

# run from root directory
uv run main.py
```
